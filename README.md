## + How to Run:
Step 1: Terminal
- Terminal: Tab 1 (ecommerce-laravel-api)
```
cd ecommerce-laravel-api
php artisan serve
```

- Terminal: Tab 2 (ecommerce-reactjs-api)
```
cd ecommerce-reactjs-api
npm run dev
```

Step 2: Run
```
http://localhost:5173/
```
